# nova-ui Example App

This is a showcase app demonstrating all nova-ui components.

## Setup

1. Create a new React Native project:

```bash
npx react-native init NovaUIExample
cd NovaUIExample
```

2. Install nova-ui:

```bash
npm install nova-native-ui
```

3. Copy the `App.tsx` from this directory into your project root, replacing the default `App.tsx`.

4. Run the app:

```bash
# iOS
npx react-native run-ios

# Android
npx react-native run-android
```

## What's Included

The example app demonstrates:

- **Theme System** — Color palette visualization
- **Button** — All variants, colors, sizes, loading, and disabled states
- **Typography** — All text variants, weights, and decorations
- **Input** — Outline, filled, underline variants with validation states
- **Card** — Elevated, outlined, and filled variants
- **Layout** — Box, Stack (HStack/VStack), and Divider
- **Avatar** — Multiple sizes with auto-generated initials
- **Badge** — Solid, outline, soft variants with BadgeDot
- **Chip** — Selectable, closable chips
- **Switch** — Animated toggle with labels
- **Checkbox** — Animated checkboxes with labels
- **RadioGroup** — Animated radio selection
- **Modal** — Dialog with form example
- **Toast** — Success, error, warning, and info notifications
